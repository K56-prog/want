# WantToDo List

## 概要
。このアプリでは、以下の機能を利用できます：
- 新しいタスクの追加
- タスクの詳細確認と編集
- タスクの削除

[デモはこちら](https://K56-prog.github.io/todo-app/)

---

## 使用技術
- Django
- HTML
- Bootstrap

---

## 機能一覧
1. タスクを追加
2. タスクの詳細確認 / 編集
3. タスクを削除

---

### ローカルで動かす場合

   '''bash

2. プロジェクトフォルダに移動

3. ブラウザでindex.htmlを開く。
